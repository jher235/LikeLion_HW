이번 과제에서 구현할 기능은 아래와 같다

1. 유저 생성
2. 유저 정보 수정
3. 유저 삭제
4. 유저 로그인
5. 유저가 메모 작성
6. 유저가 자신이 작성한 특정 메모 조회, 자신이 작성한 모든 메모 조회
7. 유저가 자신이 작성한 특정 메모 삭제
8. 유저가 자신이 작성한 특정 메모 수정
9. 유저가 모든 메모에 좋아요와 좋아요 취소 가능
10. 특정 메모에 좋아요 누른 유저들 조회
11. 유저가 소속된 단체 조회
12. 단체 생성
13. 유저가 단체 가입
14. 유저가 단체 탈퇴
15. 단체에 소속된 유저 목록 조회
